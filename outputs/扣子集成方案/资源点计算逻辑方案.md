## 1. 概述

### 1.1 背景与目标
在AI工作流平台中，用户使用不同工作流执行任务时，需要消耗相应数量的资源点（Points）。资源点系统是平台的核心经济体系，需要设计一套公平、透明、可扩展的计算逻辑，实现以下目标：

1. **公平性**：相同参数的任务消耗相同的资源点
2. **透明性**：用户能清楚了解计费规则和计算过程
3. **灵活性**：支持动态定价、促销活动等业务需求
4. **可扩展性**：适应未来新增的工作流类型和计算规则

### 1.2 设计原则
1. **成本导向**：资源点消耗与实际成本（API调用、计算资源等）挂钩
2. **用户友好**：计费规则简单易懂，避免复杂计算
3. **业务灵活**：支持多种定价策略，适应市场变化
4. **技术可行**：计算逻辑易于实现和维护

## 2. 资源点计算模型

### 2.1 核心计算公式
```
总消耗点数 = 基础点数 × 时长系数 × 分辨率系数 × 质量系数 × 折扣系数
```

### 2.2 计算因子说明

#### 2.2.1 基础点数（Base Points）
**定义**：工作流的基础定价，反映工作流类型和复杂程度

**定价策略**：
| 工作流类型 | 基础点数 | 说明 |
|-----------|---------|------|
| **文本生成** | 500 | 纯文本生成，成本较低 |
| **图像处理** | 800 | 图像编辑、风格转换等 |
| **图像生成** | 1200 | AI生图，成本较高 |
| **音频处理** | 1000 | 音频剪辑、音效添加 |
| **视频处理** | 2000 | 视频编辑、特效添加 |
| **复杂合成** | 3000 | 多模态内容合成 |

**实现方式**：
- 后台管理系统配置
- 存储在`workflows`表的`base_points`字段

#### 2.2.2 时长系数（Duration Factor）
**定义**：根据任务处理时长调整的系数

**计算规则**：
| 时长范围 | 系数 | 说明 |
|---------|------|------|
| < 30秒 | 1.0 | 快速任务，不额外计费 |
| 30秒 - 2分钟 | 1.2 | 中等时长，轻度溢价 |
| 2分钟 - 5分钟 | 1.5 | 较长时间，合理溢价 |
| 5分钟 - 10分钟 | 2.0 | 长时间任务，显著溢价 |
| > 10分钟 | 3.0 | 超长任务，高溢价 |

**实现逻辑**：
```java
public class DurationFactorCalculator {
    
    public double calculateFactor(long durationSeconds) {
        if (durationSeconds < 30) {
            return 1.0;
        } else if (durationSeconds <= 120) { // 2分钟
            return 1.2;
        } else if (durationSeconds <= 300) { // 5分钟
            return 1.5;
        } else if (durationSeconds <= 600) { // 10分钟
            return 2.0;
        } else {
            return 3.0;
        }
    }
    
    // 根据工作流类型微调
    public double calculateFactor(long durationSeconds, WorkflowType type) {
        double baseFactor = calculateFactor(durationSeconds);
        
        // 视频任务时长系数更高
        if (type == WorkflowType.VIDEO_PROCESSING) {
            return baseFactor * 1.2;
        }
        
        // 文本任务时长系数较低
        if (type == WorkflowType.TEXT_GENERATION) {
            return Math.max(1.0, baseFactor * 0.8);
        }
        
        return baseFactor;
    }
}
```

#### 2.2.3 分辨率系数（Resolution Factor）
**定义**：根据输出内容的分辨率调整的系数

**图像分辨率系数**：
| 分辨率 | 系数 | 说明 |
|-------|------|------|
| 512×512 | 1.0 | 标准分辨率 |
| 768×768 | 1.3 | 中等分辨率 |
| 1024×1024 | 1.6 | 高分辨率 |
| 2048×2048 | 2.5 | 超高分辨率 |
| 4096×4096 | 4.0 | 专业级分辨率 |

**视频分辨率系数**：
| 分辨率 | 系数 | 说明 |
|-------|------|------|
| 480p | 1.0 | 标清 |
| 720p | 1.5 | 高清 |
| 1080p | 2.0 | 全高清 |
| 2K | 3.0 | 2K分辨率 |
| 4K | 5.0 | 4K超高清 |

**实现逻辑**：
```java
public class ResolutionFactorCalculator {
    
    // 图像分辨率系数
    public double calculateImageFactor(String resolution) {
        switch (resolution) {
            case "512×512":
                return 1.0;
            case "768×768":
                return 1.3;
            case "1024×1024":
                return 1.6;
            case "2048×2048":
                return 2.5;
            case "4096×4096":
                return 4.0;
            default:
                // 解析自定义分辨率
                return parseCustomResolution(resolution);
        }
    }
    
    // 视频分辨率系数
    public double calculateVideoFactor(String resolution) {
        switch (resolution) {
            case "480p":
                return 1.0;
            case "720p":
                return 1.5;
            case "1080p":
                return 2.0;
            case "2K":
                return 3.0;
            case "4K":
                return 5.0;
            default:
                return 1.0;
        }
    }
    
    private double parseCustomResolution(String resolution) {
        // 解析如 "1920x1080" 格式的分辨率
        String[] parts = resolution.split("[×x]");
        if (parts.length == 2) {
            try {
                int width = Integer.parseInt(parts[0]);
                int height = Integer.parseInt(parts[1]);
                
                // 按像素数量估算系数
                long pixels = (long) width * height;
                long basePixels = 512L * 512L; // 262144像素
                
                return Math.sqrt((double) pixels / basePixels);
            } catch (NumberFormatException e) {
                return 1.0;
            }
        }
        return 1.0;
    }
}
```

#### 2.2.4 质量系数（Quality Factor）
**定义**：根据输出质量要求调整的系数

**计算规则**：
| 质量等级 | 系数 | 说明 |
|---------|------|------|
| 快速 | 0.8 | 低质量，快速生成 |
| 标准 | 1.0 | 标准质量，平衡速度和质量 |
| 精细 | 1.5 | 高质量，细节更好 |
| 超精细 | 2.5 | 最高质量，最佳细节 |

**特殊场景**：
- **艺术风格强化**：×1.3
- **真实感增强**：×1.4
- **高清修复**：×1.8

#### 2.2.5 折扣系数（Discount Factor）
**定义**：根据促销活动、用户等级等因素调整的折扣

**计算规则**：
| 折扣类型 | 系数 | 说明 |
|---------|------|------|
| 无折扣 | 1.0 | 原价 |
| 新用户 | 0.5 | 首次注册赠送30000点，首单半价 |
| VIP用户 | 0.9 | VIP用户享受9折优惠 |
| 促销活动 | 0.7-0.9 | 根据不同活动设置 |
| 批量优惠 | 0.8 | 一次性充值大量点数 |

## 3. 详细计算流程

### 3.1 预估计算（提交任务前）
```
┌─────────────────────────────────────┐
│     用户提交任务参数                  │
├─────────────────────────────────────┤
│ 1. 获取工作流基础点数                 │
│ 2. 解析参数，计算各项系数            │
│ 3. 计算预估消耗点数                  │
│ 4. 检查用户余额是否足够              │
│ 5. 返回预估结果给用户确认            │
└─────────────────────────────────────┘
```

**预估计算流程**：
```java
@Service
public class PointEstimationService {
    
    @Resource
    private WorkflowService workflowService;
    
    @Resource
    private FactorCalculator factorCalculator;
    
    public EstimationResult estimatePoints(String workflowId, 
                                           Map<String, Object> parameters) {
        // 1. 获取工作流信息
        Workflow workflow = workflowService.getWorkflow(workflowId);
        
        // 2. 计算各项系数
        double durationFactor = factorCalculator.calculateDurationFactor(
            parameters, workflow.getType());
        
        double resolutionFactor = factorCalculator.calculateResolutionFactor(
            parameters, workflow.getType());
        
        double qualityFactor = factorCalculator.calculateQualityFactor(
            parameters);
        
        double discountFactor = factorCalculator.calculateDiscountFactor(
            getCurrentUserId());
        
        // 3. 计算预估点数
        int basePoints = workflow.getBasePoints();
        double estimatedPoints = basePoints * durationFactor * 
                                resolutionFactor * qualityFactor * 
                                discountFactor;
        
        // 4. 四舍五入到整数
        int roundedPoints = (int) Math.round(estimatedPoints);
        
        // 5. 返回预估结果
        return EstimationResult.builder()
            .basePoints(basePoints)
            .durationFactor(durationFactor)
            .resolutionFactor(resolutionFactor)
            .qualityFactor(qualityFactor)
            .discountFactor(discountFactor)
            .estimatedPoints(roundedPoints)
            .currency("points")
            .build();
    }
}
```

### 3.2 实际计算（任务完成后）
```
┌─────────────────────────────────────┐
│     扣子平台返回执行结果               │
├─────────────────────────────────────┤
│ 1. 获取实际执行时长、输出参数          │
│ 2. 重新计算各项系数（基于实际数据）    │
│ 3. 计算实际消耗点数                   │
│ 4. 扣减用户资源点余额                 │
│ 5. 记录详细计费信息                   │
└─────────────────────────────────────┘
```

**实际计算流程**：
```java
@Service
public class PointDeductionService {
    
    @Resource
    private TaskService taskService;
    
    @Resource
    private UserPointService userPointService;
    
    @Resource
    private FactorCalculator factorCalculator;
    
    @Transactional(rollbackFor = Exception.class)
    public DeductionResult deductPoints(String taskId, 
                                        CozeExecutionResult executionResult) {
        
        // 1. 获取任务信息
        Task task = taskService.getTask(taskId);
        Workflow workflow = workflowService.getWorkflow(task.getWorkflowId());
        
        // 2. 计算实际消耗点数
        int actualPoints = calculateActualPoints(task, executionResult, workflow);
        
        // 3. 扣减用户余额
        UserPointBalance balance = userPointService.getBalance(task.getUserId());
        
        if (balance.getAvailablePoints() < actualPoints) {
            throw new InsufficientPointsException(
                "用户余额不足，需要：" + actualPoints + 
                "，可用：" + balance.getAvailablePoints());
        }
        
        // 4. 扣减点数
        balance.setAvailablePoints(balance.getAvailablePoints() - actualPoints);
        userPointService.updateBalance(balance);
        
        // 5. 记录消费记录
        PointConsumptionRecord record = createConsumptionRecord(
            task, actualPoints, executionResult);
        
        // 6. 返回扣费结果
        return DeductionResult.builder()
            .taskId(taskId)
            .userId(task.getUserId())
            .actualPoints(actualPoints)
            .balanceAfter(balance.getAvailablePoints())
            .recordId(record.getId())
            .build();
    }
    
    private int calculateActualPoints(Task task, 
                                      CozeExecutionResult result, 
                                      Workflow workflow) {
        
        // 获取实际参数
        long actualDuration = result.getActualDurationSeconds();
        Map<String, Object> actualParams = result.getActualParameters();
        
        // 计算实际系数
        double durationFactor = factorCalculator.calculateDurationFactor(
            actualDuration, workflow.getType());
        
        double resolutionFactor = factorCalculator.calculateResolutionFactor(
            actualParams, workflow.getType());
        
        double qualityFactor = factorCalculator.calculateQualityFactor(
            actualParams);
        
        double discountFactor = factorCalculator.calculateDiscountFactor(
            task.getUserId());
        
        // 计算实际点数
        double points = workflow.getBasePoints() * durationFactor * 
                       resolutionFactor * qualityFactor * discountFactor;
        
        return (int) Math.round(points);
    }
}
```

## 5. 核心算法实现

### 5.1 系数计算引擎
```java
@Component
public class FactorCalculationEngine {
    
    @Resource
    private PricingConfigRepository pricingConfigRepository;
    
    @Resource
    private DiscountRuleEngine discountRuleEngine;
    
    /**
     * 综合计算资源点消耗
     */
    public PointCalculationResult calculatePoints(
            Workflow workflow, 
            TaskExecutionData executionData) {
        
        // 1. 获取定价配置
        WorkflowPricing pricing = pricingConfigRepository
            .findActivePricing(workflow.getId());
        
        if (pricing == null) {
            throw new PricingNotFoundException(
                "工作流 " + workflow.getId() + " 未配置定价");
        }
        
        // 2. 计算各项系数
        double durationFactor = calculateDurationFactor(
            executionData.getDurationSeconds(), 
            workflow.getType(),
            pricing.getDurationFactorConfig());
        
        double resolutionFactor = calculateResolutionFactor(
            executionData.getOutputResolution(),
            workflow.getType(),
            pricing.getResolutionFactorConfig());
        
        double qualityFactor = calculateQualityFactor(
            executionData.getQualityLevel(),
            pricing.getQualityFactorConfig());
        
        // 3. 计算折扣系数
        double discountFactor = discountRuleEngine.calculateDiscountFactor(
            executionData.getUserId(),
            workflow.getId(),
            executionData.getParameters());
        
        // 4. 计算总点数
        int basePoints = pricing.getBasePoints();
        double totalPoints = basePoints * durationFactor * 
                           resolutionFactor * qualityFactor * discountFactor;
        
        // 5. 应用上下限
        int minPoints = pricing.getMinPoints();
        int maxPoints = pricing.getMaxPoints();
        
        int roundedPoints = applyLimits(totalPoints, minPoints, maxPoints);
        
        // 6. 返回计算结果
        return PointCalculationResult.builder()
            .basePoints(basePoints)
            .durationFactor(durationFactor)
            .resolutionFactor(resolutionFactor)
            .qualityFactor(qualityFactor)
            .discountFactor(discountFactor)
            .totalPoints(roundedPoints)
            .breakdown(createBreakdown(pricing, executionData))
            .build();
    }
    
    private int applyLimits(double points, int min, Integer max) {
        int rounded = (int) Math.round(points);
        
        // 应用下限
        if (rounded < min) {
            return min;
        }
        
        // 应用上限（如果配置）
        if (max != null && rounded > max) {
            return max;
        }
        
        return rounded;
    }
}
```

### 5.2 折扣规则引擎
```java
@Component
public class DiscountRuleEngine {
    
    @Resource
    private DiscountRuleRepository discountRuleRepository;
    
    @Resource
    private UserService userService;
    
    public double calculateDiscountFactor(String userId, 
                                          String workflowId, 
                                          Map<String, Object> parameters) {
        
        // 1. 获取用户信息
        User user = userService.getUser(userId);
        
        // 2. 查询适用的折扣规则
        List<DiscountRule> applicableRules = discountRuleRepository
            .findApplicableRules(user, workflowId, new Date());
        
        if (applicableRules.isEmpty()) {
            return 1.0; // 无折扣
        }
        
        // 3. 按优先级排序
        applicableRules.sort((r1, r2) -> 
            Integer.compare(r2.getPriority(), r1.getPriority()));
        
        // 4. 合并折扣（取最大折扣）
        double bestDiscountFactor = 1.0;
        
        for (DiscountRule rule : applicableRules) {
            // 检查条件是否满足
            if (checkConditions(rule, user, parameters)) {
                if (rule.getDiscountFactor() < bestDiscountFactor) {
                    bestDiscountFactor = rule.getDiscountFactor();
                }
            }
        }
        
        return bestDiscountFactor;
    }
    
    private boolean checkConditions(DiscountRule rule, 
                                   User user, 
                                   Map<String, Object> parameters) {
        
        JSONObject conditions = rule.getConditions();
        if (conditions == null || conditions.isEmpty()) {
            return true;
        }
        
        // 检查用户等级条件
        if (conditions.has("userLevel")) {
            int requiredLevel = conditions.getInt("userLevel");
            if (user.getLevel() < requiredLevel) {
                return false;
            }
        }
        
        // 检查充值金额条件
        if (conditions.has("rechargeAmount")) {
            int requiredAmount = conditions.getInt("rechargeAmount");
            if (user.getTotalRecharge() < requiredAmount) {
                return false;
            }
        }
        
        // 检查参数条件
        if (conditions.has("parameters")) {
            JSONObject paramConditions = conditions.getJSONObject("parameters");
            if (!checkParameterConditions(paramConditions, parameters)) {
                return false;
            }
        }
        
        return true;
    }
}
```

### 5.3 预扣与结算系统
```java
@Service
public class PointPreDeductionService {
    
    @Resource
    private RedisTemplate<String, String> redisTemplate;
    
    @Resource
    private PointCalculationEngine calculationEngine;
    
    // 预扣点数
    public PreDeductionResult preDeductPoints(String taskId, 
                                              Workflow workflow, 
                                              Map<String, Object> parameters) {
        
        // 1. 计算预估点数
        TaskExecutionData execData = TaskExecutionData.builder()
            .userId(getCurrentUserId())
            .parameters(parameters)
            .build();
        
        PointCalculationResult calcResult = calculationEngine
            .calculatePoints(workflow, execData);
        
        int estimatedPoints = calcResult.getTotalPoints();
        
        // 2. 锁定点数
        String lockKey = "point:lock:" + getCurrentUserId();
        long locked = redisTemplate.opsForValue().increment(lockKey, estimatedPoints);
        
        // 3. 存储预估信息
        String estimationKey = "point:estimation:" + taskId;
        redisTemplate.opsForValue().set(estimationKey, 
            JsonUtils.toJson(calcResult), 24, TimeUnit.HOURS);
        
        return PreDeductionResult.builder()
            .taskId(taskId)
            .estimatedPoints(estimatedPoints)
            .lockedPoints(locked)
            .breakdown(calcResult.getBreakdown())
            .build();
    }
    
    // 最终结算
    public SettlementResult settlePoints(String taskId, 
                                         int actualPoints) {
        
        // 1. 获取预估信息
        String estimationKey = "point:estimation:" + taskId;
        String estimationJson = redisTemplate.opsForValue()
            .get(estimationKey);
        
        // 2. 释放或调整锁定点数
        String lockKey = "point:lock:" + getCurrentUserId();
        long currentlyLocked = Long.parseLong(
            redisTemplate.opsForValue().get(lockKey));
        
        int adjustment = 0;
        if (estimationJson != null) {
            PointCalculationResult calcResult = JsonUtils.fromJson(
                estimationJson, PointCalculationResult.class);
            adjustment = calcResult.getTotalPoints() - actualPoints;
        }
        
        // 3. 调整锁定
        if (adjustment > 0) {
            // 释放多余锁定
            redisTemplate.opsForValue().decrement(lockKey, adjustment);
        } else if (adjustment < 0) {
            // 增加锁定
            redisTemplate.opsForValue().increment(lockKey, -adjustment);
        }
        
        // 4. 实际扣减
        UserPointBalance balance = userPointService.getBalance(
            getCurrentUserId());
        balance.setAvailablePoints(balance.getAvailablePoints() - actualPoints);
        userPointService.updateBalance(balance);
        
        // 5. 清理预估数据
        redisTemplate.delete(estimationKey);
        
        return SettlementResult.builder()
            .taskId(taskId)
            .actualPoints(actualPoints)
            .adjustment(adjustment)
            .finalBalance(balance.getAvailablePoints())
            .build();
    }
}
```

## 6. 动态定价策略

### 6.1 市场供需定价
```java
@Component
public class DynamicPricingEngine {
    
    @Resource
    private TaskStatisticsRepository statisticsRepository;
    
    @Resource
    private PricingAdjustmentRepository adjustmentRepository;
    
    /**
     * 根据供需情况动态调整定价
     */
    public DynamicPricingResult adjustPricing(Workflow workflow) {
        
        // 1. 获取当前供需数据
        TaskStatistics stats = statisticsRepository
            .findRecentStatistics(workflow.getId(), 24); // 最近24小时
        
        // 2. 计算供需比
        double supplyDemandRatio = calculateSupplyDemandRatio(stats);
        
        // 3. 根据供需比计算调整系数
        double adjustmentFactor = calculateAdjustmentFactor(supplyDemandRatio);
        
        // 4. 获取历史调整记录
        List<PricingAdjustment> recentAdjustments = adjustmentRepository
            .findRecentAdjustments(workflow.getId(), 7); // 最近7天
        
        // 5. 平滑调整（避免频繁大幅变动）
        double smoothedFactor = smoothAdjustment(adjustmentFactor, recentAdjustments);
        
        // 6. 应用调整
        int newBasePoints = (int) Math.round(
            workflow.getBasePoints() * smoothedFactor);
        
        // 7. 确保在合理范围内
        newBasePoints = Math.max(MIN_POINTS, Math.min(newBasePoints, MAX_POINTS));
        
        return DynamicPricingResult.builder()
            .originalPoints(workflow.getBasePoints())
            .adjustedPoints(newBasePoints)
            .adjustmentFactor(smoothedFactor)
            .supplyDemandRatio(supplyDemandRatio)
            .reason(generateAdjustmentReason(stats, supplyDemandRatio))
            .build();
    }
    
    private double calculateSupplyDemandRatio(TaskStatistics stats) {
        if (stats.getCompletedTasks() == 0) {
            return 1.0; // 默认平衡
        }
        
        // 供需比 = 待处理任务数 / 已完成任务数
        return (double) stats.getPendingTasks() / stats.getCompletedTasks();
    }
    
    private double calculateAdjustmentFactor(double ratio) {
        // 供需比越高（需求大于供给），定价越高
        if (ratio > 2.0) {
            return 1.5; // 需求远大于供给，溢价50%
        } else if (ratio > 1.5) {
            return 1.3; // 需求大于供给，溢价30%
        } else if (ratio > 1.2) {
            return 1.1; // 需求略大于供给，溢价10%
        } else if (ratio < 0.5) {
            return 0.7; // 供给大于需求，降价30%
        } else if (ratio < 0.8) {
            return 0.8; // 供给略大于需求，降价20%
        } else {
            return 1.0; // 供需平衡，保持原价
        }
    }
}
```

### 7.1 异常检测
```java
@Component
public class PointConsumptionAnomalyDetector {
    
    @Resource
    private PointConsumptionRepository consumptionRepository;
    
    @Resource
    private UserBehaviorRepository behaviorRepository;
    
    public AnomalyDetectionResult detectAnomalies(String userId, 
                                                  PointConsumptionRecord record) {
        
        List<Anomaly> anomalies = new ArrayList<>();
        
        // 1. 异常高消耗检测
        if (isUnusuallyHighConsumption(userId, record.getTotalPoints())) {
            anomalies.add(Anomaly.HIGH_CONSUMPTION);
        }
        
        // 2. 频繁消费检测
        if (isFrequentConsumption(userId)) {
            anomalies.add(Anomaly.FREQUENT_CONSUMPTION);
        }
        
        // 3. 参数异常检测
        if (hasSuspiciousParameters(record.getParameters())) {
            anomalies.add(Anomaly.SUSPICIOUS_PARAMETERS);
        }
        
        // 4. 时间异常检测
        if (isUnusualTime(record.getCreatedAt())) {
            anomalies.add(Anomaly.UNUSUAL_TIME);
        }
        
        return AnomalyDetectionResult.builder()
            .userId(userId)
            .recordId(record.getId())
            .anomalies(anomalies)
            .riskScore(calculateRiskScore(anomalies))
            .actionRequired(!anomalies.isEmpty())
            .build();
    }
    
    private boolean isUnusuallyHighConsumption(String userId, int points) {
        // 获取用户历史消费记录
        List<PointConsumptionRecord> history = consumptionRepository
            .findRecentConsumptions(userId, 30); // 最近30天
        
        if (history.isEmpty()) {
            return points > 10000; // 新用户，超过10000点视为异常
        }
        
        // 计算历史平均值和标准差
        double average = history.stream()
            .mapToInt(PointConsumptionRecord::getTotalPoints)
            .average()
            .orElse(0);
        
        double stdDev = calculateStdDev(history, average);
        
        // 超过平均值3个标准差视为异常
        return points > average + 3 * stdDev;
    }
}
```

```java
@Component
public class PointQuotaManager {
    
    @Resource
    private UserPointService userPointService;
    
    @Resource
    private ConsumptionRateLimiter rateLimiter;
    
    public QuotaCheckResult checkQuota(String userId, int requiredPoints) {
        
        // 1. 检查可用余额
        UserPointBalance balance = userPointService.getBalance(userId);
        
        if (balance.getAvailablePoints() < requiredPoints) {
            return QuotaCheckResult.insufficientBalance(
                balance.getAvailablePoints(), requiredPoints);
        }
        
        // 2. 检查消费频率
        if (!rateLimiter.checkRateLimit(userId)) {
            return QuotaCheckResult.rateLimitExceeded();
        }
        
        // 3. 检查单日消费上限
        int dailyConsumed = consumptionRepository
            .getDailyConsumption(userId, new Date());
        
        int dailyLimit = getDailyLimit(userId);
        if (dailyConsumed + requiredPoints > dailyLimit) {
            return QuotaCheckResult.dailyLimitExceeded(
                dailyConsumed, dailyLimit, requiredPoints);
        }
        
        // 4. 检查单次消费上限
        int maxSingleConsumption = getMaxSingleConsumption(userId);
        if (requiredPoints > maxSingleConsumption) {
            return QuotaCheckResult.singleLimitExceeded(
                requiredPoints, maxSingleConsumption);
        }
        
        return QuotaCheckResult.approved();
    }
}
```
