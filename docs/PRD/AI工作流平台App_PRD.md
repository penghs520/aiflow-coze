## 1. 项目概述

### 1.1 项目背景
基于扣子平台构建的AI工作流生态，用户可在扣子中发布视频创作、图文创作等各类工作流。本项目旨在开发一款跨平台移动应用（支持iOS和Android），为用户提供便捷的工作流浏览、预览、使用体验，并实现用户注册、资源点充值、任务管理等商业化闭环。

### 1.2 产品定位
- **核心价值**：降低AI工作流使用门槛，提供一站式创作工具集
- **目标用户**：内容创作者、自媒体从业者、电商运营、AI爱好者
- **竞品参考**：米核平台（商用AI模块）、即梦App（UI风格）

### 1.3 技术栈
- **前端**：React Native跨平台开发（支持iOS/Android）
- **后端**：Java/SpringBoot（团队已有能力）
- **集成平台**：扣子工作流API
- **数据库**：MySQL/Redis

### 1.4 核心指标
- 工作流上线数量：支持上百个工作流动态配置
- 用户注册转化率：首次注册赠送30000资源点
- 任务完成率：后台任务执行成功率 > 95%
- 用户体验：页面加载时间 < 2秒

## 2. 参考界面分析

### 2.1 米核平台界面要点（基于截图分析）
- **工作流列表**：网格/列表布局，显示工作流封面、名称、简介、消耗点数
- **工作流详情**：演示视频预览、参数配置面板、运行成本明细
- **参数输入**：支持文本输入、图片上传、视频上传、下拉选择等控件
- **任务状态**：任务队列、执行进度、完成通知

### 2.2 即梦App界面风格（基于截图与搜索结果）
- **整体风格**：深色主题（#121212背景，#1E1E1E卡片），冷色调点缀
- **导航结构**：底部五标签（推荐、灵感、想象、消息、我）
- **组件特征**：
  - 圆角卡片（radius=12dp）
  - 毛玻璃效果（背景模糊）
  - 悬浮操作按钮
  - 分段控制器（Segment Control）
- **字体系统**：
  - 标题：Roboto Bold 18sp
  - 正文：Roboto Regular 14sp
  - 辅助文本：Roboto Light 12sp

### 2.3 设计原则提取
1. **沉浸式浏览**：大图预览、视频自动播放、无干扰布局
2. **渐进式输入**：参数分组展示，按需展开高级选项
3. **透明化计费**：实时计算资源点消耗，明确展示成本
4. **状态可视化**：任务进度实时反馈，异常状态明确提示

## 3. 用户画像

### 3.1 核心用户群体
| 用户类型 | 特征 | 核心需求 | 使用场景 |
|---------|------|---------|---------|
| **内容创作者** | 自媒体博主、视频UP主 | 快速生成高质量内容，降低制作成本 | 每周需要产出3-5条视频/图文内容 |
| **电商运营** | 中小商家、电商团队 | 商品图修图、营销素材生成 | 大促期间批量处理上千张产品图 |
| **AI爱好者** | 技术尝鲜者、学生 | 体验最新AI能力，学习工作流搭建 | 探索不同工作流效果，用于个人项目 |
| **企业用户** | 市场部、设计部 | 团队协作、统一资源管理 | 多成员使用，需要充值管理和任务分配 |

### 3.2 用户旅程地图
```
发现工作流 → 预览效果 → 输入参数 → 确认成本 → 提交任务 → 查看进度 → 获取结果
```

## 4. 功能需求

### 4.1 工作流展示模块（优先级：最高）

#### 4.1.1 工作流发现
- **列表视图**：网格/列表切换，支持排序（最新、热门、消耗点数）
- **分类筛选**：按类型（视频创作、图文创作）、标签、难度筛选
- **搜索功能**：关键词搜索，支持工作流名称、简介、标签
- **收藏功能**：用户可收藏常用工作流，快速访问

#### 4.1.2 工作流详情
- **预览区域**：演示视频自动播放（静音），图文示例轮播
- **基本信息**：工作流名称、作者、更新日期、使用次数、平均评分
- **功能描述**：详细说明工作流功能、适用场景、输出示例
- **参数预览**：简要展示需要输入的参数类型（无需展开具体表单）

#### 4.1.3 个性化推荐
- **基于行为**：根据用户历史使用记录推荐相似工作流
- **热门榜单**：实时热门工作流排行（使用次数、收藏数）
- **新手上路**：新手专属推荐（简单、低消耗工作流）

### 4.2 参数输入与任务提交模块（优先级：高）

#### 4.2.1 参数输入界面
- **动态表单**：根据工作流定义动态生成输入控件
- **输入类型**：
  - 文本输入框（单行/多行）
  - 图片上传（从相册选择或拍照，支持多张）
  - 视频上传（从相册选择或录制）
  - 下拉选择（单选/多选）
  - 滑块（数值范围）
  - 开关（布尔值）
- **参数分组**：基础参数（必填）与高级参数（可选，默认折叠）
- **实时验证**：输入时即时验证格式、大小、数量限制

#### 4.2.2 视频参数专项设置
- **清晰度**：标清、高清、超清（对应不同资源点消耗）
- **比例**：9:16、16:9、1:1、4:3等预设比例
- **时长**：输入期望时长（秒），或选择预设时长
- **帧率**：24fps、30fps、60fps（高级选项）

#### 4.2.3 资源点计算与确认
- **实时计算**：根据输入参数、视频设置实时计算预计消耗点数
- **明细展示**：拆解计算逻辑（基础消耗 + 参数加成）
- **余额显示**：当前可用资源点数，预估剩余点数
- **确认弹窗**：提交前再次确认参数与消耗，支持修改

#### 4.2.4 任务提交
- **一键运行**：提交任务到后台队列
- **异步处理**：任务进入后台执行，用户可继续浏览其他工作流
- **排队提示**：显示当前排队任务数，预估等待时间

### 4.3 任务中心模块（优先级：中）

#### 4.3.1 任务列表
- **状态分类**：进行中、已完成、失败、已取消
- **基本信息**：任务ID、工作流名称、提交时间、预计完成时间
- **进度展示**：实时进度条（0%-100%），当前执行阶段描述
- **操作入口**：查看详情、取消任务（仅进行中）、重新运行（失败时）

#### 4.3.2 任务详情
- **输入参数回顾**：展示提交时的参数快照
- **执行日志**：分阶段日志，错误信息高亮显示
- **结果展示**：生成的文件预览（图片/视频），下载按钮
- **资源点结算**：实际消耗点数（可能与预估有差异）

#### 4.3.3 通知系统
- **推送通知**：任务完成、任务失败、资源点不足时推送
- **消息中心**：历史通知列表，支持一键清空

### 4.4 用户系统模块（优先级：低）

#### 4.4.1 注册与登录
- **手机号注册**：验证码登录/注册
- **第三方登录**：微信、QQ快速登录（可选）
- **首次注册奖励**：自动赠送30000资源点，弹窗提示

#### 4.4.2 个人中心
- **账户信息**：头像、昵称、手机号、注册时间
- **资源点管理**：当前余额、充值记录、消耗记录
- **我的收藏**：收藏的工作流列表，快速入口
- **设置**：通知开关、主题切换（深色/浅色）、清除缓存

#### 4.4.3 充值系统
- **套餐选择**：固定金额套餐（如30元/10000点），自定义金额
- **支付方式**：微信支付、支付宝支付
- **到账提示**：支付成功后实时到账，余额更新

### 4.5 后台管理模块（独立Web系统）

#### 4.5.1 工作流管理
- **上架/下架**：控制工作流在前端的可见性
- **参数配置**：定义工作流需要的参数类型、默认值、验证规则
- **资源点定价**：设置基础消耗点数，参数加成系数
- **版本管理**：工作流更新迭代，支持多版本并存

#### 4.5.2 用户管理
- **用户列表**：查看注册用户，禁用/启用账户
- **充值审核**：人工审核大额充值（可选）
- **数据统计**：用户增长、活跃度、留存率报表

#### 4.5.3 任务监控
- **实时队列**：查看正在执行的任务，强制取消异常任务
- **日志查询**：按时间、用户、工作流筛选执行日志
- **性能监控**：API调用成功率，平均执行时长，错误率报警

## 5. 非功能需求

### 5.1 性能要求
- **页面加载**：冷启动 < 3秒，页面切换 < 1秒
- **图片加载**：列表图片懒加载，详情图片预加载
- **视频预览**：首帧加载 < 2秒，流畅播放（缓存机制）
- **API响应**：关键接口 P95 < 500ms

### 5.2 兼容性要求
- **移动平台**：支持 iOS 13+ 和 Android 8.0（API 26）及以上
- **屏幕适配**：适配主流分辨率（1080p、2K、全面屏、刘海屏）
- **网络环境**：支持弱网环境，断点续传（大文件上传）

### 5.3 安全性要求
- **数据传输**：HTTPS加密，敏感信息脱敏
- **用户认证**：Token机制，定期刷新
- **支付安全**：支付密码/生物识别二次验证
- **防刷机制**：接口限流，验证码防机器操作

### 5.4 可用性要求
- **易学性**：新用户5分钟内完成首次任务提交
- **效率**：常用工作流3步内完成参数输入
- **错误恢复**：操作失败明确提示，提供恢复路径
- **一致性**：遵循Material Design规范，保持交互统一

## 7. 交互设计说明

### 7.1 核心交互流程

#### 7.1.1 工作流使用流程
```
进入首页 → 浏览/搜索工作流 → 点击进入详情 → 观看预览 → 点击"立即使用" →
进入参数输入页 → 填写参数 → 实时计算资源点 → 确认提交 → 跳转任务中心 →
等待执行 → 收到通知 → 查看结果
```

#### 7.1.2 参数输入交互
- **分组折叠**：高级参数默认折叠，点击展开
- **实时反馈**：输入时立即验证，错误信息即时显示
- **示例提示**：复杂参数旁提供示例按钮，点击查看示例值
- **保存草稿**：长时间输入支持自动保存草稿，断网可恢复

#### 7.1.3 任务状态交互
- **进度可视化**：环形进度条，阶段提示文字
- **异常处理**：失败任务提供"查看原因"按钮，一键重新提交
- **结果预览**：图片直接展示，视频支持缩略图+播放按钮

### 7.2 手势与动画
- **下拉刷新**：工作流列表、任务列表支持下拉刷新
- **上拉加载**：列表分页加载，显示加载动画
- **页面切换**：右滑返回（iOS）、返回按钮（Android），页面过渡动画
- **按钮反馈**：点击涟漪效果，操作成功/失败动画

### 7.3 空状态与错误态
- **空列表**：图标+提示文字+引导按钮（如"去发现工作流"）
- **网络错误**：重试按钮，离线缓存提示
- **参数错误**：高亮错误项，聚焦并提示正确格式
- **资源点不足**：充值引导弹窗，提供快速充值入口

## 8. 优先级排序与版本规划

### 8.1 MVP版本（V1.0）
**核心目标**：跑通工作流展示 → 参数输入 → 任务提交核心闭环

**包含功能**：
1. 工作流列表与详情（静态数据）
2. 基础参数输入（文本、单张图片）
3. 资源点计算与任务提交
4. 任务中心基础查看
5. 用户注册登录（手机验证码）
6. 后台工作流配置（基础版）

**排除功能**：
1. 视频专项设置（清晰度、比例、时长）
2. 多图/视频上传
3. 收藏功能
4. 充值系统（使用赠送点数）
5. 个性化推荐

**核心目标**：完善创作体验，支持复杂参数

2. 多文件上传（图片批量、视频）
3. 收藏与历史记录
4. 充值系统与支付

### 8.3 扩展版本（V3.0）
**核心目标**：个性化与商业化深化

**新增功能**：
1. 个性化推荐算法
